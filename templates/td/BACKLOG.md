# Backlog

Session-scoped parking. Append items mid-session to defer them; flushed to GitHub Issues (with the appropriate type) at `/td-close`. Starts empty each session, ends empty each session.

For mid-session flush without waiting for `/td-close`: just say "park the backlog to GH" — the items get triaged into GH Issues one-by-one with dedupe checks.
