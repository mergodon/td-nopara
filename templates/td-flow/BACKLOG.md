# Backlog

Session-scoped parking. Append items mid-session to defer them; flushed to GitHub Issues (with the appropriate type) at `/td-flow-close`. Starts empty each session, ends empty each session.

For mid-session flush without waiting for `/td-flow-close`: just say "park the backlog to GH" (or run `/td-flow-park`) — related lines are consolidated, deduped against open issues, and batch-created with type in one decision pass.
